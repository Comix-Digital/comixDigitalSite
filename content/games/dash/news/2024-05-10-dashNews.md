HELLO PEOPLE!

These are the changes in Dash v0.8. This version is currently only for web  as I am working on a save progress system that I want to finish before releasing this update for Windows and Mac.
- Added jumping but only for the...
- 5 new levels I also made
- Removed the "GOOD JOB!" text on Level 5
- Added a level select screen

The level select screen will be fully unlocked on the web version which means you'll be able to play any level you want right out of the gate. When I release the update for Windows and Mac you will have to unlock each level by completing the previous one. As I mentioned before your progress will save so you will not have to re-beat levels you have already completed.
